title: Get all Clans
---

```js
GET https://narutodb.xyz/api/clan
```

curl -X 'GET' \
        'https://narutodb.xyz/api/clan' \
     -H 'accept: application/json'

#### Pagination and Limit

```js
https://narutodb.xyz/api/clan?page=1&limit=20
```

  'https://narutodb.xyz/api/clan?page=1&limit=20' \
  -H 'accept: application/json'
